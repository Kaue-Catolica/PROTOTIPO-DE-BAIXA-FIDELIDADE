Introdução

A linguagem C foi criada em 1972 por Dennis Ritchie, cientista da computação da Bell Labs, para desenvolver o sistema operacional Unix. A linguagem C é resultado da evolução da linguagem B, que por sua vez se baseou na linguagem BCPL.

A linguagem C é considerada uma linguagem de nível intermediário, pois combina elementos de linguagens de alto nível com funcionalidades de linguagens de nível máquina.

Alguns pontos importantes da história da linguagem C são: A versão mais utilizada da linguagem é a ANSI C, C89 ou C90, publicada entre 159 e 1989.

Em 1999, a ISO publicou o padrão ISO/IEC 9899:1999, conhecido como C99. Em 2011, foi publicado o padrão mais recente, o C11. A linguagem C inspirou a criação de outras linguagens de programação, como C++, Java, C#, JavaScript, Perl, PHP, e Lua.
